# QueueForge Backend

This folder contains the Spring Boot backend for QueueForge.

The backend will provide:

- JWT authentication
- User, organization, and project APIs
- Queue management APIs
- Job creation APIs
- Worker coordination APIs
- Retry and Dead Letter Queue logic
- Metrics APIs
- Swagger/OpenAPI documentation

## Planned Stack

- Java
- Spring Boot
- Spring Security
- PostgreSQL
- Flyway
- JPA / Hibernate
- JUnit
- Testcontainers
